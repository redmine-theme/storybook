# Redmine Storybook

Redmine のコンポーネントを Storybook で表示します。

## Usage

```
$ npm install
$ npm run storybook
```

## Theme develop

In your theme directory.

```
$ npm install -D @redmine-theme/storybook
```

Create `.storybook/main.js` `.storybook/preview.js` `.storybook/preview-head.html`

```
$ npm run storybook
```
